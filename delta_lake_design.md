# [Best Practise] Delta Lake Storage Configuration

The following is an example directory structure for a Medallion Architecture Data Lake contained within a single cloud storage account.



```json
data_lake_cloud_storage
│
├── bronze_zone (Ingestion Zone)
│   ├── domain_name_1
│   │    ├── delta_table (Delta table)
│   │    ├── csv_table(CSV table)
│   │    ├── parquet_table (Parquet table)
│   │    ├── ...
│   │    └── ...
|   ├── domain_name_2
│   │    ├── delta_table (Delta table)
│   │    ├── csv_table (CSV table)
│   │    ├── parquet_table (Parquet table)
│   │    ├── ...
│   │    └── ...
|   ├── ...
|   └── ...
├── silver_zone (Trusted Zone)
│   ├── domain_name_1
│   │    ├── sub_domain_1 (optional)
│   │    │   ├── delta_table (Delta table)
│   │    │   ├── ...
│   │    │   └── ...
│   │    ├── sub_domain_2 (optional)
│   │    │   ├── delta_table (Delta table)
│   │    │   ├── ...
│   │    │   └── ...
│   │    ├── delta_table (Delta table)
│   │    ├── ...
│   │    └── ...
|   ├── domain_name_2
│   │    ├── sub_domain_1 (optional)
│   │    │   ├── delta_table (Delta table)
│   │    │   ├── ...
│   │    │   └── ...
│   │    ├── sub_domain_2 (optional)
│   │    │   ├── delta_table (Delta table)
│   │    │   ├── ...
│   │    │   └── ...
│   │    ├── delta_table (Delta table)
│   │    ├── ...
│   │    └── ...
├── gold-zone (Aggregate Zone)
│   ├── domain_name_1
│   │    ├── sub_domain_1
│   │    │   ├── project_1
│   │    │   │   ├── delta_table (Delta table)
│   │    │   │   ├── ...
│   │    │   ├── ...

│   │    │   ├── ...

│   │    ├── sub_domain_2 (optional)
│   │    │     ├── delta_table (Delta table)
│   │    │     ├── ...
│   │    │     └── ...
│   │    ├── delta_table (Delta table)
│   │    ├── ...
│   │    └── ...

│   ├── bronze-zone
 ```
